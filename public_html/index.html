Some settings you might wanna do with zmud are:

1) Change the colors in: View / Preferences / Colors to this:

Line 1:        | black
Line 2: white  | black
Line 3:  red   | black
Line 4: yellow | black
Line 5: black  | white
Line 6: green  | black

2) Under: View / Preferences / Special Characters:
-Enable _ALL_ characters there. Example Pic.

3) View / Preferences / General, find the section "Script Parser"
-Make sure the only things _NOT_ checked are:
*Show Triggers
*Echo Commands (leave checked if you want to see local_echo)
*Echo Scripts
*Wildcards in {} triggers. Example Pic.
